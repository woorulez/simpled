# simpled

a simple HTTP file server

## how to install
* using `go get`
  ```bash
  go get -u github.com/woorulez/simpled
  ```
* pre-built binaries are at [releases](https://github.com/woorulez/simpled/releases)

## how to use
```
$ ./simpled -h
Usage of ./simpled:
  -d string
    	directory (default ".")
  -p int
    	port (default 11011)
```
